## What it is

A GitHub Action that counts real **repository traffic** (views and clones from
the official GitHub Traffic API) and renders a customisable SVG **badge**
committed to a dedicated branch in your own repository.

The Action runs entirely inside your own GitHub account on the free Actions
runner. There is no hosted service, no external account, and no cost to either
the maintainer or you.

## What it is not

It is **not** a profile-views counter. GitHub does not expose any profile-views
API, and image-based "profile views" badges (proxied through GitHub's Camo
image cache) are systematically undercounted and effectively unverifiable.
This project is the honest alternative for **repositories**.
